To install PyEvolve, first download the package from this link: 

http://pyevolve.sourceforge.net/0_6rc1/intro.html#windows


For Python 2.7, choose Pyevolve v.0.6rc1 (source code) for Python 2.x

Unzip the package inside your Python\Lib\Site-packages folder. 
Via command window, run the setup.py file with the command "python setup.py install"
